## Usage
### Install Dependencies (frontend & backend)
- npm install
- cd frontend
- npm install

### Run
Run frontend (:3000) & backend (:5000) 
- npm run dev

Run backend only 
- npm run server

### Build & Deploy
Create frontend prod build
- cd frontend
- npm run build

### Seed Database

Import data 
- npm run data:import

Destroy data 
- npm run data:destroy
